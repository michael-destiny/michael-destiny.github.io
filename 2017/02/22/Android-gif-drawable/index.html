Android_gif_drawable

android-gif-drawable项目中使用的gif库.

Gif文件格式说明:

可以参考 http://blog.csdn.net/wzy198852/article/details/17266507

github: https://github.com/koral--/android-gif-drawable.git

涉及文件:

java/pl/droidsonroids/gif/GifDrawable.java

java/pl/droidsonroids/gif/GifInfoHandle.java

jni/gif.c

jni/gif.h

jni/dgif_lib.c

jni/open_close.c

入口 GifDrawable.

在创建GifDrawable对象的时候，需要传入GifInfoHandle类型的参数.不同的gif来源的图片经过GifInfoHandle的统一处理，生成统一的GifInfoHandle对象。

GifDrawable() -> GifInfoHandle.openXXX() -> pl_droidsonroids_gif.so里的native functions.<br>

gif.c中，可以看到以上调用的native函数.

Java_pl_droidsonroids_gif_GifInfoHandle_openFile()
Java_pl_droidsonroids_gif_GifInfoHandle_openFd()
Java_pl_droidsonroids_gif_GifInfoHandle_openByteArray()
Java_pl_droidsonroids_gif_GifInfoHandle_openDirectByteBuffer()
Java_pl_droidsonroids_gif_GifInfoHandle_openStream()
Java_pl_droidsonroids_gif_GifInfoHandle_openFd()

创建的函数都是统一构建GifSourceDescriptor

最终将gifHandle进行返回。

createGifHandle() -> DDGifSlurp() -> info->rewindFunction(info);
在这次返回之前，先读取了Gif首帧具体数据之前的所有内容.

Header:
    Version, ScreenWidth, ScreenHeight, 全局颜色列表标志, 颜色深度, 像素宽高比
    全局颜色列表.

Content:
    ImageDescriptor, 局部颜色列表, 基于颜色列表的图象数据,
    图形控制扩展(optional), CommentExtension(optional)....

Trailer:
    GIF Trailer - 标识GIF文件结束，固定值0x3B

在读取完Content[0].图像数据头以后，完成初步的Gif文件的解析。

在rewind结束以后，目前所处的位置在读取完整个header以后的位置. 即descriptor.startPos.

回到GifDrawable()构造函数中，我们开始继续drawable的构造.

GifDrawable的构造中线程控制在GifRenderingExecutor的单例当中。

而GifDrawable的渲染则在RenderTask中实现。

gifDrawable.invalidateSelf()这里在确认中.
